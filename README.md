# Knowledge-Management-Notes

A personal wiki built with [Obsidian](https://obsidian.md) and versioned with Git.

This vault is my long-term knowledge base — a growing collection of concepts,
references, and notes connected to my development and learning journey.

It covers topics I explore professionally and personally, from knowledge management
and documentation practices to technical concepts and tools I work with day to day.

## Structure

- `/concepts` — definitions, frameworks, and reference material
- `/projects` — notes tied to active work or side projects
- `/learning` — courses, books, talks, and key takeaways
- `/reference` — templates, checklists, and reusable resources

## Why Git

Storing the vault here keeps it portable, version-controlled, and accessible
across machines — without relying on any cloud platform.
